【プログラム名】
　しげぴよの制作部屋

【概要】
　究極二択くん　：ユーザーが投稿した2択質問を次々と回答していき簡易性格診断を表示するゲーム
　しりとり師範くん：ユーザーが投稿した画像と読み方を蓄積し、画像のみの選択肢を頼りにしりとりをつなげていくゲーム

【ローカルでの使用方法】
　・run_local_manager.bat をダブルクリックしてください。
　　(Python 3.07以降がないと起動できません。下記参照。)
　・「テスト」タブの「ローカル起動」ボタンからローカル上でテスト実行できます。
　・ローカル起動(テスト)を初めて使う場合は、下記ボタンを押してください。
　　「初期セットアップ(モジュールインストール)」ボタン(初めの1回のみ)。
　・「DB」管理タブの「オンライン版で上書き」ボタン(学習データの同期)。

【ローカル起動できない場合】
・Windows11のSmartScreenでブロックされる場合は、以下の手順をお試しください。
　-　run_local_manager.batを右クリックし、「プロパティ」を開く。
　-　「全般」タブの右下にあるセキュリティの項目の 「許可する」（または「ブロックの解除」）にチェックを入れる。
　-　「適用」を押してプロパティを閉じてから、再度実行する。

【しりとり師範くんの投稿画像が表示されない場合】
　-　ローカルテスト時は投稿画像のオンラインURLを直接参照しますが、万が一うまく読み込めない場合は、
　　　https://sigepiyo338.pythonanywhere.com/media/media.zip
　　をダウンロード・解凍しmediaフォルダをsiritori-main直下に配置してみてください。
　　(ただし、26/07/14現在までのものです)

【動作環境】
　Python 3.07 以降必須
　Google Chrome必須
　Windows 11 (25H2) 64bitでのみ動作確認済

　・Python 3.07 以降 がない場合
　　https://www.python.org/downloads/windows/
　　　　の「Download Windows installer...」で大丈夫だと思います。
